# Simple JSP Web Project

## 📌 Description
This is a basic web project using JSP, HTML, CSS, and JavaScript.  
It includes a simple UI with a button click action and a greeting form.

---

## 📁 Files Included
- index.jsp → Main webpage
- style.css → Styling for the page
- script.js → JavaScript functionality

---

## 🚀 Features
- Button click message display
- User input form
- Dynamic greeting message
- Simple responsive UI

---

## 🛠️ How to Run
1. Install Apache Tomcat server
2. Place project inside `webapps` folder
3. Start Tomcat server
4. Open browser and go to:
   http://localhost:8080/your_project_folder/index.jsp
